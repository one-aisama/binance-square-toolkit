# Правила контента

- Контент генерирует АГЕНТ, не софт. Софт предоставляет `ContentGenerator` и `CommentGenerator` как инструменты которые агент вызывает.
- Правила контента определены в `config/content_rules.yaml` — агент следует им при генерации.
- Весь контент на английском языке.
- Стиль: разговорный, человеческий, как обычный человек. Никаких AI-клише (см. banned_phrases в content_rules.yaml).
- Посты: минимум 2 абзаца, $CASHTAGS для монет, #хэштеги.
- Комменты: 1-2 предложения максимум, релевантные конкретному посту, как разговор с автором. Осмысленные, цепляющие.
- Цитаты (quote reposts): $CASHTAGS только если оригинальный пост про конкретную монету. Добавить свою позицию.
- Использовать ТОЛЬКО реальные рыночные данные из API — никогда не выдумывать цены или проценты.
- AI API вызовы используют провайдер/модель из settings.yaml.
