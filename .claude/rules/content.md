# Content Rules

- Content is generated by the AGENT, not the software. The software provides `ContentGenerator` and `CommentGenerator` as tools that the agent calls.
- Content rules are defined in `config/content_rules.yaml` — the agent follows them during generation.
- All content is in English.
- Style: conversational, human, like a regular person. No AI cliches (see banned_phrases in content_rules.yaml).
- Posts: minimum 2 paragraphs, $CASHTAGS for coins, #hashtags.
- Comments: 1-2 sentences max, relevant to the specific post, like a conversation with the author. Meaningful, engaging.
- Quotes (quote reposts): $CASHTAGS only if the original post is about a specific coin. Add your own take.
- Use ONLY real market data from the API — never fabricate prices or percentages.
- AI API calls use the provider/model from settings.yaml.
