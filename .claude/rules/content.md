# Content Rules

- Content is generated by the AGENT, not the code. The code provides `ContentGenerator` and `CommentGenerator` as tools the agent can call.
- Content rules are defined in `config/content_rules.yaml` — the agent follows them when generating.
- All content in English.
- Style: conversational, human, like a real person. No AI cliches (see banned_phrases in content_rules.yaml).
- Posts: minimum 2 paragraphs, $CASHTAGS for coins, #hashtags.
- Comments: 1-2 sentences max, relevant to the specific post, like a conversation with the author. Thoughtful, engaging.
- Quote reposts: $CASHTAGS only if the original post is about a specific coin. Add your own take.
- Use ONLY real market data from API — never fabricate prices or percentages.
